# Steps
1. Initialise project
```console
npm init -y
```
2. install dependencies
```console
npm i -D typescript @types/express @types/node 
```
3. Initialise typescript config file
```console
npx tsc --init
```
4. Keep folder structure same as normal backend folder
- public
	- temp
- src
	- config
	- controllers
	- middlewares
	- models
	- routes
	- types
	- utils
	- index.ts (file)
	- server.ts (file)
